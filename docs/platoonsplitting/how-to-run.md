# How to Run the Getting Started Example

## Prerequisites

Before running this simulation, ensure you have the following installed:

- **OMNeT++** (v6.0 or later) - The network simulator framework
- **SUMO** (v1.0 or later) - Simulation of Urban MObility traffic simulator
- **Plexe** - (v3.2) The platooning extension for Veins (already included in this project)
- **Veins** - (v5.3.1) Vehicular Network Simulation framework
- **Python 3** - For build configuration and scripting
- **C++ compiler** - GCC or Clang (for building the simulation)
